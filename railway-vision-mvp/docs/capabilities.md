# RVision Capabilities

## Real executable capabilities
- Asset intake: image, video, ZIP dataset upload and reuse.
- Model validation: create inference tasks with released or selected models.
- Training preparation: OCR review, state review, dataset export, training job creation.
- Model lifecycle: candidate submission, approval, release preparation, evidence export.
- Device and audit operations: inspect device status, release scope, and audit trails.
- Data governance: preview and run cleanup actions from settings.

## Capabilities that require Expert Console
- Fine-grained training parameter control.
- Pipeline registration and release details.
- Device authorization details.
- Audit and governance execution.

## Capability boundaries for AI
- AI can recommend pages, prefill parameters, and explain missing information.
- AI should not promise actions that do not exist in the current UI or permissions.
- AI must route users to Expert Console when the task needs precise manual control.
